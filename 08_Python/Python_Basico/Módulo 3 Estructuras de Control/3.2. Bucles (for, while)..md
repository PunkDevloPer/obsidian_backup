siguiendo con las estructuras de control, una parte importante del código escrito por la humanidad debe llevar tareas repetitivas y constantes

aquí es donde entran los bucles que nos ayudan a ejecutar una tarea de forma repetitiva o semiautomática
el bucle For va a ejecutar la tarea una vez y ya, mientras el bucle while va a ejecutar la tarea de forma infinita a  menos que una condición indique lo contrario
````python
list =("josep","juan","carlos")
for count in list:
	 print(count)
```
